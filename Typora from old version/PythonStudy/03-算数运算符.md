## 算术运算符

算术运算符是运算符的一种，用来处理四则运算

| 运算符 |  描述  |     实例      |
| :----: | :----: | :-----------: |
|   +    |   加   | 10 + 20 = 30  |
|   -    |   减   | 10 - 20 = -10 |
|   *    |   承   | 10 * 20 = 200 |
|   /    |   除   | 10 / 20 = 0.5 |
|   //   | 取整除 |  9 // 2 = 4   |
|   %    |  取余  |   9 % 2 = 1   |
|   **   |   幂   |  2 ** 3 = 8   |

- 在python中， `*`  可用于字符串，表示重复字符串

```python
a = 'nihao  '
print(a * 10)
```

- 可以用 `+` 拼接字符串
