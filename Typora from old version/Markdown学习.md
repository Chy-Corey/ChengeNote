# Markdown学习

## 标题

“# ” + “空格”作为前缀，一个井号代表一级，最高有六级标题

## 字体

**粗体**：文本两边加“**”

*斜体*：文本两边加“*”

***斜体加粗***：文本两边加“***”

~~删除线~~：文本两边加“~~”

## 引用

> “>” + “空格”作为前缀

## 分割线

单独的“---”（三个减号）构成分割线

---

## 图片

“!“ + ”[图片名字]“ + ”()”

## 超链接

“[链接名字]” + “(链接地址)”

[煜宝的博客]([Liberty](http://101.34.51.13/))

## 列表

### 有序列表

“数字” + “.” + “空格”

1. 
2. 

### 无序列表

”减号“ + ”空格“

- 
-
